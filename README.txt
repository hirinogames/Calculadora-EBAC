O código do calculadora.sh foi feito para ser usado no linux e utiliza o python 3.

Passo a passo de como executar:

-Salve o arquivo em uma pasta acessível pelo seu terminal linux;
-No file armazenado, use o comando "chmod 744 calculadora.sh" para tornar o arquivo executável;
-Use o comando "\. calculadora.sh" para executar o código.

Explicação do código:

-O código recebe dois números, podem ser diferentes ou iguais, atenção, a ordem dos números afeta o resultado. Ex: 7 - 2 = 5 e 2 - 7 = -5;
-A calculadora pode executar as 4 operações básicas: somar, subtrair, multiplicar e dividir;
-Caso queira alterar os números digitados, digite 5 na seleção de operação;
-Caso queira encerrar a execução do código, digite 0 na seleção de operação;
-A calculadora pede uma nova operação após executar a selecionada;
-Caso digite algo fora das opções, uma mensagem de erro irá aparecer e a calculadora pedirá para selecionar uma operação.
